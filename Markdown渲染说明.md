## 📋 支持的 Markdown 格式

### 1️⃣ **文本格式**

- **加粗文本**：使用 `**文本**` 或 `__文本__`
- *斜体文本*：使用 `*文本*` 或 `_文本_`
- ***粗斜体***：使用 `***文本***`
- `行内代码`：使用反引号

### 2️⃣ **列表**

#### 无序列表：
- 项目一
- 项目二
  - 子项目 2.1
  - 子项目 2.2
- 项目三

#### 有序列表：
1. 第一步
2. 第二步
3. 第三步

### 3️⃣ **标题**

支持 H1-H6 六级标题：

# H1 一级标题
## H2 二级标题
### H3 三级标题

### 4️⃣ **链接**

[点击访问百度](https://www.baidu.com)

### 5️⃣ **引用**

> 这是一段引用文本
> 可以多行显示

### 6️⃣ **代码块**

```python
def hello_world():
    print("Hello, World!")
```

## 🎨 自定义样式

我为不同的 Markdown 元素配置了专门的样式：

| 元素 | 样式 |
|------|------|
| **加粗** | `font-bold text-slate-900` |
| *斜体* | `italic` |
| `代码` | 灰色背景，圆角 |
| 链接 | 蓝色，悬停下划线 |
| 列表 | 带缩进和圆点/数字 |
